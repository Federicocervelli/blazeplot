# BlazePlot v0.3.8

## Changes

- Fixed on-demand rendering for streaming examples and previews by routing live appends through `SeriesStore`, so each append marks the chart dirty and schedules a render.
- Added OHLC append/update helpers on `SeriesStore` for on-demand-safe candlestick/OHLC streams.
- Fixed the website home-page streaming charts and live/server-sampled previews so they continue updating without requiring pan/zoom interaction.
