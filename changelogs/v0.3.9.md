# BlazePlot v0.3.9

## Changes

- Added the generic series mutation API for live data: `series.append({ x, y })`, `series.append({ y })`, `series.append({ x, open, high, low, close })`, and `series.updateLast({ open, high, low, close })`.
- Added `series.replace(...)` for replaceable datasets such as `ServerSampledDataset`.
- Updated website previews to exercise generic appends with on-demand rendering.
- Marked legacy append helpers as deprecated for the next migration path: `series.append(x, y)`, `series.appendY(...)`, `series.appendOhlc(...)`, and `series.updateLastOhlc(...)` remain supported but now point to the generic object-based APIs.
