# BlazePlot v0.3.3

## Changes

- Added `ServerSampledDataset` plus `downsample: "server"` for rendering server-provided point samples or min/max buckets without applying a second client-side sampler.
- Added a server-sampled Binance preview with public REST kline buckets and a live 5s candlestick chart aggregated from the public aggTrade WebSocket.
- Improved trackpad interactions: smoother pinch zoom, two-finger panning, and preserved mouse-wheel zoom behavior.
- Added `OhlcRingBuffer.updateLast()` for updating the currently forming live candle in place.
- Added chart screenshot/export improvements, fit-to-data and auto-follow policies, accessibility/focus improvements, plugin ARIA semantics, annotation hit events, SVG overlay screenshots, package/bundle validation checks, and documentation guides.
