# BlazePlot v0.3.6

## Changes

- Added the optional `blazeplot/plugins/flamegraph` WebGL flame graph/flame chart/status-span plugin, including folded-stack parsing, model builders, hover/click picking, search highlighting, labels, screenshots, and website preview coverage.
- Improved large-trace rendering with data-anchored tiny-frame aggregation and minimum visible frame dimensions so zoomed-out flame charts remain visible and pan smoothly.
- Stabilized dense LOD sampling for panning/live-follow charts by anchoring buckets to zoom/data rather than the moving viewport edge.
- Improved hover tooltip/marker responsiveness and drag behavior for dense interactive charts.
- Made outside axes the default and expanded outside gutters when axis titles are present.
