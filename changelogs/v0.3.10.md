# BlazePlot v0.3.10

## Changes

- Added row-array series appends for convenient moderate-rate feeds, including XY, implicit-X, and OHLC row batches.
- Added in-place series update helpers: `series.updateLast(...)` and `series.updateAt(...)` for XY, implicit-X, and OHLC/candlestick streams.
- Added fixed-rate stream shorthand: `chart.addLine({ capacity, xStart, xStep })` creates an implicit-X `UniformRingBuffer` so callers can append with `series.append({ y })`.
- Added `chart.followLatestX(...)`, `chart.stopFollowingLatestX()`, follow-state helpers, and smooth clock-based follow via `currentX` for timestamped real-time streams.
- Updated built-in interactions so double-click/tap reset resumes live follow by default, with `resumeFollowOnReset: false` available when reset should stay paused.
- Improved live-follow behavior across missing data gaps and epoch timestamp domains.
- Improved linear tick generation so the right side is not truncated when a nice step would otherwise produce too many ticks.
- Added an irregular dense IoT sensor stream preview with real time axes, batched appends, missing-data gaps, spikes, and live-follow behavior.
- Added live-data documentation and improved streaming troubleshooting/API guidance.
