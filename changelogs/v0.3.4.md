# BlazePlot v0.3.4

## Changes

- Replaced the previous rendering backend dependency with the native WebGL2 backend and kept the deprecated `ReglBackend` compatibility alias.
- Added lean public subpath exports for core, render, interaction, linked-core, data, export, and built-in plugins to keep chart-only imports small.
- Added data export helpers for visible ranges, selections, CSV/JSON serialization, and lightweight transforms.
- Improved line/area gap semantics, WebGL context restoration, min/max line draw batching, and backend projection handling.
- Split screenshot composition into a lazy-loaded chunk and trimmed published package sourcemap payload.
- Expanded bundle-size reporting, README bundle documentation, first-frame/core footprint notes, and version-pinned library comparison docs.
- Refreshed roadmap/docs and made the preview favicon transparent.
