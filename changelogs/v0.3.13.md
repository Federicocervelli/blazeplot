# BlazePlot v0.3.13

## Changes

- Added first-class histogram helpers, chart APIs, documentation, tests, and preview coverage.
- Improved axis tick label placement so endpoint labels stay readable and overlapping labels are suppressed.
- Improved crosshair overlays with interval-aware highlights, configurable label placement, and line rendering below chart content.
- Added a large-scale 1M-bin histogram website preview with an initial 1k-bin viewport.
